NRMLC: Nesting Recursive Model Logic Computer

Inventor: Martin Bradford Hovsepian Jr.
Email: nestingrealmsmodel@gmail.com

Overview

NRMLC is a symbolic computing system that replaces binary logic with phase-based recursion, signal strength, and decaying memory.
